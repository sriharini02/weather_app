# 🌦️ Weather App with Machine Learning

## 📌 Overview

This project is a **Machine Learning-based web application** that predicts the impact of weather conditions on transportation systems. It helps in identifying potential delays or disruptions based on environmental factors such as temperature, humidity, rainfall, and wind speed.

The application is built using **Django** for the backend and **Machine Learning models** for prediction, providing both analytical insights and visual representations of weather patterns.


## 🚀 Features

* Predicts transportation disruptions based on weather conditions
* Trained on **15,000+ real-world records**
* Achieves **87% accuracy** using ML models
* Interactive web interface built with Django
* Data visualization using Seaborn heatmaps
* Supports multiple weather parameters for prediction

## ⚙️ How It Works

1. **Data Collection**

   * Dataset with 15,000+ records including weather parameters and transport status.

2. **Data Preprocessing**

   * Handling missing values
   * Cleaning and structuring data
   * Feature selection

3. **Model Training**

   * Logistic Regression (for classification)
   * Decision Tree (for improved prediction accuracy)

4. **Prediction**

   * User inputs weather conditions
   * Model predicts transportation impact (Delay / No Delay)

5. **Visualization**

   * Heatmaps and graphs to analyze weather patterns



## 🧠 Machine Learning Models Used

* Logistic Regression
* Decision Tree Classifier

**Model Accuracy:** 87%

## 📊 Example Use Case

A user enters weather details like:

* Temperature: 32°C
* Humidity: 85%
* Rainfall: Heavy

👉 The system predicts whether transportation will be affected and provides insights for decision-making.

## ▶️ Installation & Setup

1. Clone the repository:

```

```

2. Navigate to the project folder:

```
cd your-repo-name
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the server:

```
python manage.py runserver
```

5. Open in browser:

